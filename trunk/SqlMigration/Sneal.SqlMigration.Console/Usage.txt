Sneal.SqlMigration.Console.exe

Scripts the specified SQL Server objects to disk.  This utility will script
complete or differential SQL scripts.

If a target server and db are specified, then differential SQL scripts are 
created which can be used to upgrade the target db to match the source db.

Example Usage:

sneal.sqlmigration.console /dir=d:\exportdir /data /multiplefiles /tables=WorkOrder;Customer /server=localhost /db=AdventureWorks2000

sneal.sqlmigration.console /schema /tables=WorkOrder;Customer /server=localhost /db=AdventureWorks2000 /target-server=localhost /target-db=AdventureWorks2000
